O que incluir no pedido

Para ajudar no atendimento, envie modelo do iPhone, versão do iOS, versão do app, tela ou fluxo em que o problema aconteceu, passos para reproduzir e mensagem de erro exibida, se houver.

Se o problema envolve Tempo de Uso, informe se a permissão foi concedida no iOS.

Não envie dados sensíveis

Não envie CPF, IMEI, senhas, códigos de verificação, conteúdo de mensagens, documentos pessoais, nomes de bancos usados, dados de cartão, chaves Pix ou credenciais de qualquer serviço.

Limites conhecidos

  • FamilyControls, ManagedSettings e seletor de apps exigem permissão de Tempo de Uso e entitlement aprovado para distribuição.
  • A validação completa desses recursos exige dispositivo real.
  • A build atual não solicita localização.
  • Compras digitais só devem aparecer quando produtos StoreKit estiverem configurados no App Store Connect.
  • O app não oferece compra externa para recursos digitais.

Rolê Seguro é uma ferramenta preventiva de privacidade em locais públicos. Ele não é serviço de recuperação de aparelho e não substitui recursos oficiais do iOS, Apple, governo, bancos ou operadoras.
